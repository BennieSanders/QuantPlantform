# Quant Platform

一个基于 Vue3 + FastAPI + Python 的量化交易平台课程项目。

## 项目目标
实现一个基础量化交易平台，支持：
- 用户登录
- 股票/标的选择
- 策略选择
- 回测执行
- 回测结果展示
- 后续扩展 AI 策略推荐

## 技术架构
- Frontend: Vue3 + Vite
- Backend: FastAPI
- Quant Engine: Python
- Database: MySQL
- Chart: ECharts

## 项目结构
- frontend: 前端页面
- backend: 后端服务
- quant-engine: 策略与回测核心
- docs: 项目文档
- scripts: 启动脚本
- data: 示例数据
